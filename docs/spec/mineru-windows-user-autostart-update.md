# MinerU Windows 用户级自启动与稳定更新：需求规格

## 一句话描述

为使用 PaperLens 本地 MinerU 薄服务的 Windows 用户提供无需管理员权限的登录自启动和稳定通道自动更新；继续使用现有 `%LOCALAPPDATA%` 运行时、配置与 token。

## 核心功能

- 以当前用户、有限权限注册 Windows 任务计划程序登录任务，登录后隐藏启动本地服务；安装、修复和卸载时可幂等维护任务。
- 登录启动前最多每 24 小时检查一次固定的 `SWUSTcyt/paperlens` GitHub Releases 稳定通道；预发布版本不进入自动更新。
- 下载版本化 MinerU 服务 ZIP 及 SHA-256，校验后复用候选 generation 安装；候选验证失败时保留旧运行时。
- 服务已运行时跳过更新且不终止任务；提供任务状态、手动检查和立即更新命令，并输出不含 token、绝对用户路径或远端响应正文的可读诊断。

## 边界条件

不创建 Windows SCM 服务，不要求管理员权限，不迁移 `%LOCALAPPDATA%`/token 模型，不更新 Chrome 扩展，不支持任意仓库/下载 URL、测试通道、GUI 或 Docker。自动更新不负责首次安装；无网络、Release/资产缺失、哈希不匹配或服务正在运行时继续使用现有运行时。

## 非功能性与技术约束

沿用 PowerShell 5.1、uv、Python 3.12 和版本化运行时；服务仍只监听 `127.0.0.1`。任务、更新状态和下载缓存只能写入带 PaperLens 标记的用户目录。Release 仓库、资产命名和 24 小时间隔为代码常量；下载设超时和大小上限，ZIP 解包拒绝路径穿越。任何失败都不得覆盖配置、泄漏 token、切换到未验证 generation 或中断正在运行的服务。

## 接口契约

用户通过 PowerShell 脚本注册、查询、立即运行或移除登录任务，并通过更新脚本执行 `CheckOnly` 或 `UpdateNow`。成功返回退出码 0 和稳定状态码；参数、信任校验或更新失败返回非零退出码与脱敏中文消息。计划任务调用同一维护入口，先按间隔尝试更新，再以前台子进程方式运行稳定启动器。

## P0 验收总纲

有限权限任务可真实注册、登录触发配置可核验并完整移除；服务已运行时绝不更新或重复启动；只有固定稳定 Release 的版本化 ZIP 在 SHA-256、大小和安全解包均通过后才可进入候选安装；所有失败保留旧运行时和用户数据。扩展本体、全局 Python/Anaconda/CUDA、SCM 与管理员权限均不受影响。
